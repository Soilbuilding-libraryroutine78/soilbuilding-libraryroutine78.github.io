🌐 Gozar - Smart Control for Your AI Models

Download Gozar Here

Gozar acts as a central hub for your AI needs. It manages how you talk to different AI providers, routes your requests, and maintains backups if one service stops working. You keep full control of your data since Gozar runs on your own hardware.

🚀 Why Use Gozar

Using multiple AI services often leads to messy code and wasted time. Gozar simplifies this by acting as a bridge. It follows the standard rules used by OpenAI, which means most apps that work with OpenAI products will talk to Gozar without extra changes.

  • Provider Routing: Direct your requests to the best AI model for the job automatically.
  • Built-in Fallbacks: If your primary AI service fails, Gozar keeps things running by switching to a backup.
  • Usage Controls: Monitor your costs and set limits to avoid overspending on token usage.
  • Better Security: Store your API keys in one safe place instead of inside every app you use.

🛠 System Requirements

Before you start, make sure your computer meets these needs for the best experience:

  • Operating System: Windows 10 or Windows 11.
  • Memory: At least 8GB of RAM.
  • Storage: 2GB of available hard drive space.
  • Software: Docker Desktop for Windows must be installed and running.

📦 How to Install and Start

Follow these steps to set up the software on your machine:

  1. Visit the official Gozar download page to grab the latest files.
  2. Click the green Code button and select Download ZIP.
  3. Extract the folder to a location you can easily find, like your Documents or Desktop folder.
  4. Open Docker Desktop on your computer and make sure it shows a green running status.
  5. Open your Command Prompt or PowerShell window.
  6. Type cd followed by a space, then drag your extracted Gozar folder into the window. Press Enter.
  7. Type docker-compose up and press Enter.

💡 Managing Your Settings

Once you run the command, Gozar starts a server on your local machine. You can access the dashboard through your web browser. Type http://localhost:8080 into your address bar to see your controls. From here, you can input your private API keys for different services and choose which models you want to use.

🛡 Data and Privacy

Everything stays within your local network. Gozar does not send your private keys or user data to external servers. Because you host the software yourself, you decide who gets access. This setup offers the best protection for your proprietary data while giving you the speed and power of modern AI tools.

❓ Troubleshooting Common Issues

If you see an error when running the command, check these items:

  • Ensure Docker Desktop is active. The application will not work without it.
  • Verify that no other program is using port 8080. If another app uses that port, you may see an "Address already in use" error.
  • Check your internet connection if you see issues fetching the initial components.
